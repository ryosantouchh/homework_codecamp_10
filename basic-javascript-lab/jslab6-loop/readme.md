2. โจทย์ที่ได้รับ

    1. เลขที่ถูก alert เป็นลำดับสุดท้ายคือเลขอะไร
        ```
        let i = 3;

        while (i) {
            alert(i--);
        }
        ```


    2. code ทั้ง 2 อันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่

        1.
        ```
        let i = 0;
        while (++i < 5) {
            alert(i);
        }
        ```

        2.
        ```
        let i = 0;
        while (++i < 5) {
            alert(i);
        }
        ```

    
    3. code ทั้ง 2 อันนี้ จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่


        1.
        ```
        for (let i=0; i < 5; i++) alert(i); 
        ```

        2.
        ```
        for (let i=0; i < 5; ++i) alert(i); 
        ```

        ```
        for (let i=0; i<3; i++) {
            alert (`number ${i}!`);
        ```
